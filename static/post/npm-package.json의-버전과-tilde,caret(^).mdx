이 글은 아래 블로그를 읽은 뒤
글을 참고하지않고 머릿속에 남아있는 지식들을 토대로 나만의 언어로 나름대로 설명해본 것입니다.

[umanking blog](https://umanking.github.io/2022/05/05/npm-version-tilde-caret/)

**핵심 : **
^ : minor 버저닝
~ : patch 버저닝
`major.minor.patch`
1.0이하에서는 둘다 patch 버저닝

**example 문제 풀어보기**

- ~1.2.3: `>=1.2.3`, `<1.3.0`
- ^1.2.3: `>=1.2.3`, `<2.0.0`

~1.2.3이면, 1.2.2 버전은 설치되지않는 것도 유의
